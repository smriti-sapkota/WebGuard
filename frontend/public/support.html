Support

Troubleshooting help for common issues.

Backend does not start

Activate your virtual environment and install backend dependencies from requirements.txt.

Scan fails

Use browser DevTools network tab to inspect the failed API request and error response.

CORS error

Confirm your FastAPI app has CORSMiddleware configured for your frontend origin.

Empty results

Run a scan first and wait until status is complete before opening results.
